This is a great list of resources for learning languages we don't (yet!) teach, compiled originally by Patrick. **Have others? Post them below!**


C/C++  
-----
[C++ Tutorial from cplusplus.com][7]
[C and C++ Tutorials from cprogramming.com][8]
[Learn C the Hard Way][9]
[TheNewBoston's C++ Tutorials][10]
[Learn C interactive tutorial][33]
[Dev-HQ][34]

C#  
----
[Microsoft Developer Network (MSDN) C# Tutorials][11]
[MSDN Channel 9 C# Fundamentals Tutorials][12]
[C# Station Tutorial][13]

Objective-C (iOS and Mac)  
-----------
[Objective-C from Cocoa Dev Central][14]
[Apple's Objective-C intro][15] (you'll need an Apple Developer account, it's free)
[Stanford CS193P][16] Look under "more from Stanford" for iOS 5 updated versions. Also covers MVC basics that some have expressed interest in.
[TheNewBoston's Objective-C Tutorials][17]

Haskell  
------
[Learn You A Haskell][18]
[Try Haskell][19]

Java  
----
[Stanford CS106A][20] Awesome class. It covers Java but more importantly it covers program design theory that many people have asked for. It can be followed up with CS106B and CS107 also available on that website, for more theory combined with (non-Java) practical.
[TheNewBoston's Java Tutorials][21]
[Oracle's Java Tutorials][22]
[Practice-it Java practice][23]

PHP  
---
[Now on Codecademy!](http://www.codecademy.com/tracks/php)
[PHP Bounce (Interactive in the vein of Codecademy)][24]
[PHP Academy][25]
[TheNewBoston's PHP Tutorials][26]

VB.NET  
------
[MSDN Channel 9 VB Fundamentals Tutorials][27]

SQL  
---
[Stanford's Introduction to Databases via Coursera][28]
[SQL The Hard Way][35]
[W3 Schools][37]

Multiple Languages

[Programr (too many to list)][29]
[CodingBat (Java and Python)][30]

[Marakana's Youtube channel (more advanced but great, too much to list)][32]
